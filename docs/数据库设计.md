## 📊 关系型数据库设计（PostgreSQL/MySQL）

### 用户表 (users)
```sql
CREATE TABLE users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  username VARCHAR(50) UNIQUE NOT NULL,
  email VARCHAR(100) UNIQUE,
  phone VARCHAR(20),
  avatar_url TEXT,
  nickname VARCHAR(50),
  vip_level INTEGER DEFAULT 0,
  total_score INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 剧集表 (dramas)
```sql
CREATE TABLE dramas (
  id VARCHAR(50) PRIMARY KEY,  -- 如 'tianxia'
  title VARCHAR(100) NOT NULL,
  description TEXT,
  cover_url TEXT,
  total_episodes INTEGER,
  status VARCHAR(20) DEFAULT 'ongoing',  -- ongoing, completed
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 集数表 (episodes)
```sql
CREATE TABLE episodes (
  id VARCHAR(50) PRIMARY KEY,  -- 如 'tianxia-ep1'
  drama_id VARCHAR(50) REFERENCES dramas(id),
  episode_number INTEGER NOT NULL,
  title VARCHAR(100),
  video_url TEXT NOT NULL,
  subtitle_url TEXT,  -- SRT文件路径
  duration INTEGER,  -- 时长（秒）
  view_count INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 高潮点表 (climax_points)
```sql
CREATE TABLE climax_points (
  id VARCHAR(100) PRIMARY KEY,  -- 如 'ep1-climax-1'
  episode_id VARCHAR(50) REFERENCES episodes(id),
  time_start INTEGER NOT NULL,  -- 开始时间（秒）
  time_end INTEGER NOT NULL,    -- 结束时间（秒）
  time_str VARCHAR(20),         -- 如 '1:16'
  climax_type VARCHAR(50) NOT NULL,  -- 如 '柳如烟出场'
  intensity INTEGER CHECK (intensity BETWEEN 1 AND 10),
  description TEXT,
  key_dialogue TEXT,
  interaction_config JSONB,  -- 互动配置
  metadata JSONB,  -- 其他元数据
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 用户互动记录表 (user_interactions)
```sql
CREATE TABLE user_interactions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  episode_id VARCHAR(50) REFERENCES episodes(id),
  climax_point_id VARCHAR(100) REFERENCES climax_points(id),
  score INTEGER NOT NULL,
  clicks INTEGER,  -- 点击次数
  completion_time FLOAT,  -- 完成用时（秒）
  is_completed BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 索引
CREATE INDEX idx_user_interactions_user ON user_interactions(user_id);
CREATE INDEX idx_user_interactions_episode ON user_interactions(episode_id);
```

### 用户观看进度表 (watch_progress)
```sql
CREATE TABLE watch_progress (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  episode_id VARCHAR(50) REFERENCES episodes(id),
  last_position INTEGER NOT NULL,  -- 上次播放位置（秒）
  completed BOOLEAN DEFAULT false,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(user_id, episode_id)
);
```

## 🔴 实时数据设计（Redis）

### 弹幕数据
```typescript
// Redis 数据结构
interface DanmakuMessage {
  id: string
  videoId: string      // 视频ID
  time: number         // 时间点（秒）
  content: string      // 弹幕内容
  userId: string       // 用户ID
  username: string     // 用户名
  color: string        // 颜色
  position: number     // 位置（0=滚动，1=顶部，2=底部）
  createdAt: number    // 时间戳
}

// Redis 存储格式
// Key: danmaku:{videoId}
// Type: Sorted Set (按时间排序)
// Score: time (秒)
// Value: JSON字符串

// 示例
ZADD danmaku:tianxia-ep1 12.5 '{"id":"xxx","content":"柳如烟！","userId":"user1"}'
ZRANGEBYSCORE danmaku:tianxia-ep1 10 15  // 获取10-15秒的弹幕
```

### 在线用户
```typescript
// Key: online:{videoId}
// Type: Set
// Value: userId
SADD online:tianxia-ep1 user123
SCARD online:tianxia-ep1  // 获取在线人数
```

### 实时互动统计
```typescript
// Key: interaction:stats:{climaxPointId}
// Type: Hash
HSET interaction:stats:ep1-climax-1 total_clicks 12345
HSET interaction:stats:ep1-climax-1 unique_users 678
HINCRBY interaction:stats:ep1-climax-1 total_clicks 1
```

## 🚀 云服务推荐

### 方案1：Supabase（推荐起步）

**优势**：
- 免费额度大（500MB数据库 + 1GB存储）
- PostgreSQL + 实时订阅
- 内置用户认证
- 简单易用

**价格**：
- 免费版：足够开发和小规模使用
- Pro版：$25/月，8GB数据库 + 100GB存储

**使用示例**：
```typescript
import { createClient } from '@supabase/supabase-js'

const supabase = createClient(
  'https://your-project.supabase.co',
  'your-anon-key'
)

// 存储弹幕
const { data, error } = await supabase
  .from('danmaku')
  .insert({
    video_id: 'tianxia-ep1',
    time: 12.5,
    content: '柳如烟！',
    user_id: user.id
  })

// 实时订阅弹幕
const subscription = supabase
  .channel('danmaku')
  .on('postgres_changes', 
    { event: 'INSERT', schema: 'public', table: 'danmaku' },
    (payload) => {
      console.log('新弹幕:', payload.new)
    }
  )
  .subscribe()
```

### 方案2：PlanetScale + Upstash

**PlanetScale**（MySQL）：
- 适合结构化数据
- 自动扩展
- 分支功能

**Upstash**（Redis）：
- 适合实时数据
- 按请求计费
- 无服务器

### 方案3：阿里云/腾讯云（生产环境）

**阿里云**：
- RDS MySQL/PostgreSQL
- Redis
- OSS 对象存储
- CDN 加速

**腾讯云**：
- 云数据库 MySQL
- 云数据库 Redis
- COS 对象存储
- CDN

### 3. 数据库服务

```typescript
// src/lib/db.ts
import { supabase } from './supabase'

// 弹幕服务
export const danmakuService = {
  // 添加弹幕
  async add(data: {
    videoId: string
    time: number
    content: string
    userId: string
  }) {
    const { data: result, error } = await supabase
      .from('danmaku')
      .insert(data)
      .select()
    
    if (error) throw error
    return result
  },

  // 获取弹幕
  async getByTimeRange(videoId: string, startTime: number, endTime: number) {
    const { data, error } = await supabase
      .from('danmaku')
      .select('*')
      .eq('video_id', videoId)
      .gte('time', startTime)
      .lte('time', endTime)
      .order('time')
    
    if (error) throw error
    return data
  },

  // 实时订阅
  subscribe(videoId: string, callback: (danmaku: any) => void) {
    return supabase
      .channel(`danmaku:${videoId}`)
      .on('postgres_changes',
        { event: 'INSERT', schema: 'public', table: 'danmaku' },
        (payload) => callback(payload.new)
      )
      .subscribe()
  }
}

// 互动服务
export const interactionService = {
  // 记录互动
  async record(data: {
    userId: string
    episodeId: string
    climaxPointId: string
    score: number
    clicks: number
    completionTime: number
  }) {
    const { data: result, error } = await supabase
      .from('user_interactions')
      .insert(data)
      .select()
    
    if (error) throw error
    
    // 更新用户总分
    await supabase.rpc('increment_user_score', {
      user_id: data.userId,
      score_add: data.score
    })
    
    return result
  },

  // 获取用户互动记录
  async getUserInteractions(userId: string) {
    const { data, error } = await supabase
      .from('user_interactions')
      .select(`
        *,
        climax_points (*),
        episodes (*)
      `)
      .eq('user_id', userId)
      .order('created_at', { ascending: false })
    
    if (error) throw error
    return data
  }
}
```
