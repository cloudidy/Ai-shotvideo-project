## 🚀 部署说明

### 环境要求

- Node.js >= 20.x
- npm >= 10.x
- Python 3.8+（用于AI脚本）
- Whisper模型（音频转字幕）

### 快速开始

```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量（可选）
# 复制 .env.example 为 .env.local
# 添加豆包API Key

# 3. 启动开发服务器
npm run dev

# 4. 访问应用
# 打开 http://localhost:3000
```

### AI脚本使用

```bash
# 生成高光点（需要Python环境）
python scripts/generate_highlights.py tianxia 1

# 一键处理新剧（视频→音频→字幕→高光点）
python scripts/full_pipeline.py tianxia
```
