## 🛠️ 使用方法

### 1. 导入组件

```tsx
import ClimaxInteractionContainer from '@/components/ClimaxInteractionContainer'
```

### 2. 在视频播放器中使用

```tsx
'use client'

import { useState } from 'react'
import VideoPlayer from '@/components/VideoPlayer'
import ClimaxInteractionContainer from '@/components/ClimaxInteractionContainer'

export default function Episode1Page() {
  const [currentTime, setCurrentTime] = useState(0)

  const handleTimeUpdate = (time: number) => {
    setCurrentTime(time)
  }

  const handleInteractionComplete = (point: any, score: number) => {
    console.log(`完成互动: ${point.title}, 得分: ${score}`)
    // 这里可以保存到后端或本地存储
  }

  return (
    <div className="relative">
      {/* 视频播放器 */}
      <VideoPlayer
        url="/video/天下第一纨绔/第1集.mp4"
        onTimeUpdate={handleTimeUpdate}
      />

      {/* 高潮点互动 */}
      <ClimaxInteractionContainer
        currentTime={currentTime}
        onInteractionComplete={handleInteractionComplete}
      />
    </div>
  )
}
```

### 3. 配置文件位置

```
src/config/
├── episode1-interactions.ts  # 第1集互动配置
├── interactions.ts           # 互动类型配置
└── highlights.ts             # 高光点配置
```

## 🎨 组件说明

### ClimaxInteractionContainer
容器组件，管理所有的互动点。

**Props:**
- `currentTime: number` - 当前视频时间（秒）
- `onInteractionComplete?: (point, score) => void` - 互动完成回调

### ClimaxInteractionButton
单个互动按钮组件。

**Props:**
- `point: EpisodeInteractionPoint` - 互动点配置
- `currentTime: number` - 当前视频时间
- `onComplete: (point) => void` - 完成回调
- `onTimeout: (point) => void` - 超时回调

---

## 🎮 互动流程

1. **视频播放到高潮点** → 自动弹出互动按钮
2. **观众快速点击** → 进度条增长
3. **达到目标点击数** → 触发完成特效 + 奖励
4. **超时未完成** → 互动失败，继续播放

## 🔧 自定义配置

### 修改互动点

编辑 `src/config/episode1-interactions.ts`:

```typescript
{
  id: 'custom-point',
  time: 120,  // 时间点（秒）
  timeStr: '2:00',
  type: '自定义类型',
  title: '自定义标题',
  description: '自定义描述',
  intensity: 8,
  requiredClicks: 15,
  duration: 10,
  reward: {
    score: 200,
    effect: '自定义特效',
    sound: '/sounds/custom.mp3',
  },
  metadata: {
    danmakuCount: 100,
    keyDialogue: '关键台词',
    climaxType: '高潮类型',
  },
}
```

### 添加新的互动类型

编辑 `src/config/interactions.ts`:

```typescript
'new-type': {
  emoji: '🎯',
  title: '新类型',
  colors: ['#color1', '#color2'],
  flashColor: '#flash',
  danmakuTexts: ['弹幕1', '弹幕2'],
},
```

## 📝 注意事项

1. **时间同步**: 确保视频时间与互动点时间一致
2. **性能**: 大量粒子效果可能影响性能
3. **移动端**: 点击区域需要适配触摸屏
4. **音效**: 需要准备对应的音效文件
