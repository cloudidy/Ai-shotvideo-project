# 静态资源目录

## 目录结构

```
public/
├── sounds/          # 音效文件
│   ├── click.mp3    # 点击音效
│   ├── hit.mp3      # 打脸音效
│   └── power.mp3    # 逆袭音效
├── test-video.mp4   # 测试视频
└── README.md
```

## 使用说明

1. 将测试视频放入 `public/` 目录，命名为 `test-video.mp4`
2. 将音效文件放入 `public/sounds/` 目录
3. 启动项目后访问 http://localhost:3000
